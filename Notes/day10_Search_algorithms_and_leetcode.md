## Key Notes
- Linear search:
  - Check each element one by one.
  - Time complexity: O(n).
- Binary search:
  - Works on sorted lists.
  - Repeatedly divides search space in half.
  - Time complexity: O(log n).
- Binary search is much faster for large sorted data.
- LeetCode problems often combine search with careful edge-case handling.
