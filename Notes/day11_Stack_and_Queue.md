# Day 11 – Stack & Queue Problems

## Topics Covered
- Stack data structure (LIFO)
- Queue data structure (FIFO)
- Basic problems using stacks and queues

## Key Notes
- Stack:
  - Last In, First Out.
  - Operations: `push`, `pop`, `peek`.
- Queue:
  - First In, First Out.
  - Operations: `enqueue`, `dequeue`, `front`.
- Can be implemented using lists or `collections.deque` in Python.
- Used in many algorithms (DFS, BFS, parentheses checking, etc.).

## Example Concepts / Pseudo-Code
- Checking balanced parentheses using a stack.
- Simulating queue for processing tasks in order.
- Using a list or deque to simulate stack/queue operations.
