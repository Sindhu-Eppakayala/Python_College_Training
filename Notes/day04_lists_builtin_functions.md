# Day 4 – Lists & Built-in Functions

## Topics Covered
- Python lists: creation, indexing, slicing
- Common built-in list functions and methods
- Problem solving using lists and their methods

## Key Notes
- Lists are ordered, mutable collections: `[1, 2, 3]`.
- Indexing starts at 0; negative indexing allows access from the end.
- Slicing: `list[start:end:step]` to get sublists.
- Common list methods:
  - `append()`, `insert()`, `remove()`, `pop()`
  - `sort()`, `reverse()`
  - `len()` to get length.
- Lists are used for storing sequences of values and iterating over them.
