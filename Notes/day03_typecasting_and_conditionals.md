- Type casting between different data types

## Key Notes
- Strings in Python have methods like `.upper()`, `.lower()`, `.title()` to change case.
- Type casting:
  - `int("5")` converts string to integer.
  - `float("3.14")` converts string to float.
  - `str(10)` converts number to string.
- It is important to cast input data from `input()` (which returns `str`) to the correct type before operations.
- Conditionals can be combined with type casting to validate user input and handle corner cases.
