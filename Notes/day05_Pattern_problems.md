# Day 5 – Pattern Printing Problems

## Topics Covered
- Pattern printing using nested loops
- Shapes: triangle, square, rectangle, reverse triangle, right-angle triangle, diamond, timer symbol, etc.

## Key Notes
- Patterns are usually solved using nested loops: outer loop for rows, inner loop for columns.
- We use spaces and special characters (like `*` or numbers) to form shapes.
- Important concepts:
  - Controlling number of spaces vs stars.
  - Understanding row and column relationships.

## Example Concepts / Pseudo-Code
- Program to print a right-angle triangle:
  - For each row, print row number of `*`.
- Program to print a square of size `n`:
  - For each row, print `n` stars.
- Program to print a diamond shape by combining two triangles.

## What I Practiced
- Writing nested loops for different pattern shapes.
- Adjusting conditions to control where spaces and stars appear.
- Reading problem statements carefully to match exact required patterns.

## Reflection
- Pattern problems improved my understanding of loops and indexes.
- They also trained me to think clearly in terms of rows and columns.
- I want to practice more complex patterns to become faster and more accurate.
